# gameJavaScript

## Pré-requisitos.

* [nodeJS](https://nodejs.org/en/)

## Instalação

* Para instalar os plugins necessários.

``` bash
cd gameJavaScript
```

* Execute

```
npm install
```

* Para executar o código

```
yarn dev
```

* Abra o navegador de sua preferência em

```
localhost:3000
```
